## Quick publish (GitHub Pages)
1. Create a new repo (e.g. `resume-site`)
2. Upload the 3 files above into the repo root
3. Repo → Settings → Pages → Deploy from branch (main / root)

## Print to PDF
Open the site in Chrome → Print → Save as PDF.
